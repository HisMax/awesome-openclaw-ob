## 关键洞察

### 1. 从爆发增长到稳态增长

Q2 的 Stars 增速放缓不是坏事。一个健康的开源项目增长曲线应该是"S 型"的——OpenClaw 正处于 S 曲线的"减速上行"阶段，说明核心用户已经沉淀。

### 2. Fork 增速超过 Star 增速

Forks 增长 20% 而 Stars 增长 12%，Fork/Star 比率从 Q1 的 19.5% 提升到 20.9%。这意味着更多人不只是"点赞"，而是真正 fork 代码进行二次开发——社区正在从"围观者"变成"建设者"。

### 3. 流量结构变化

网站月访问量 3,800 万 vs 月活用户 320 万，人均月访问约 12 次。这说明活跃用户是高频使用者，而非一次性访客。

### 4. npm 生态的"冰山效应"

77 万周下载量中，很大一部分来自 CI/CD 自动化安装和 Docker 构建流程。实际人类安装数可能在 10-20 万/周的量级，但这恰恰说明 OpenClaw 正被纳入企业级基础设施流水线。

### 5. ClawHub 的双刃剑

13,000+ 技能带来了生态繁荣，也带来了供应链安全挑战。这与 npm 早期的安全问题如出一辙——规模越大，攻击面越广。
