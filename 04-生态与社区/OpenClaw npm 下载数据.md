# OpenClaw npm 下载数据

OpenClaw 的 npm 包下载量呈现爆发式增长，是衡量社区热度的核心指标之一。

## 周下载量趋势

| 时间 | npm 周下载量 |
|------|-------------|
| 2026.1 底（发布初期） | ~720,000 |
| 2026.2.17 | ~970,000 |
| 2026.2.25 | ~1,200,000 |
| 2026.2.26 | ~1,270,000 |
| 2026.5-6（Q2） | **~777,000** |

Q2 的周下载量从高峰有所回落，但仍维持在 77 万+。值得注意的是，大量下载来自 CI/CD 自动化安装和 Docker 构建流程，实际人类安装数估计在 10-20 万/周——这恰恰说明 OpenClaw 正被纳入企业级基础设施流水线。

- 最新版本：**openclaw 2026.6.1**（2026 年 6 月发布）
- 依赖项目数：**127+**（其他项目将 OpenClaw 作为基础设施引用）

这一增长速度与 [[OpenClaw 生态系统形成速度|生态系统形成速度]] 高度吻合。详细 Q2 数据见 [[OpenClaw GitHub 数据更新 2026Q2]]。

## 其他安装渠道

- **Homebrew**: `openclaw-cli` 30 天安装 **3,284 次**
- 累计总下载量超过 **135 万次**（Q1 数据，Q2 已远超此数）

下载量的持续飙升也是 [[OpenClaw 投资机会因素|投资机会]] 评估中的关键数据支撑。该项目基于 MIT License 发布，采用 TypeScript 开发。

## 交叉验证

npm 下载量数据与 [[OpenClaw GitHub 数据分析|GitHub Stars]] 的增长趋势基本一致，但 npm 数据更能反映真实使用量（相比 Stars 更难造假）。Discord 社区成员数也与下载量增长呈正相关。商业模式虽然尚未建立，但用户基数的爆发为未来的商业化奠定了基础。
