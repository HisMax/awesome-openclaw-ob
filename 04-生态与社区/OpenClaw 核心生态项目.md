# OpenClaw 核心生态项目


![[assets/ecosystem-projects.jpg]]

OpenClaw 的官方与社区核心项目构成了完整的生态体系。

## 官方项目

| 项目 | 描述 |
|------|------|
| **openclaw** | 核心框架（TypeScript 开发） |
| **ClawHub** | [[ClawHub 官方技能注册表|官方技能注册表]] |
| **openclaw-ansible** | Tailscale VPN + Docker 自动化部署 |
| **nix-openclaw** | Nix 包管理支持 |

## 社区精选项目

| 项目 | 描述 | Stars |
|------|------|-------|
| **awesome-openclaw-skills** | 精选 5,400+ 个技能集合 | 22K |
| **awesome-openclaw-usecases** | 真实使用案例集 | — |
| **openclaw-android** | Android 一键运行 | — |
| **openclaw-studio** | Web 管理面板 | — |
| **Paw** | 桌面应用（无需终端） | — |

这些项目共同构成了 [[OpenClaw 生态系统商业机会|生态商业机会]] 的基础。MIT License 许可使得社区项目可以自由发展和商业化。
