# ClawCon 全球大会

## 一句话总结

> 一群穿龙虾服装的开发者聚在一起展示 AI Agent——ClawCon 证明 OpenClaw 不只是工具，而是一场文化运动。

## 核心内容

ClawCon 是首个 OpenClaw 社区线下展示与交流大会。首次大会实现 "incredible turnout"，参与者穿龙虾服装展示各自的 AI Agent 项目。随后发展为全球巡回模式，在主要城市举办线下 meetup，由 [[AAIF 基金会|AAIF]] 协助组织。

## 详细分析

**大会形式**：展示环节（Showcase）让开发者现场演示 Agent 配置；龙虾服装传统延续吉祥物 Molty 的文化符号；社区黑话贯穿全场——"Yee-claw"（龙虾式打招呼）、"Claw abiding citizens"（守法公民龙虾版）。

**社区基础**：[[OpenClaw Discord 社区]] 已有 **176,000+** 成员（服务器名 "Friends of the Crustacean"），**1,200+** GitHub 贡献者。命名风波被重新诠释为"蜕壳成长"的积极叙事。

**行业意义**：一个仅发布 **2 个月**的开源项目就能举办全球大会，创下社区组建速度纪录。对比 Kubernetes（KubeCon 在项目发布 1 年后才首办），ClawCon 证明 OpenClaw 已从工具变成平台、从项目变成运动。

## 关键洞察

龙虾服装已成为 ClawCon 的标志性文化符号（详见 [[龙虾吉祥物文化]]），这不是噱头——它是社区成员表达"我属于这里"的方式。这种文化凝聚力加上 [[OpenClaw Foundation 5 天募集 $35,000|社区自发捐款 $35,000]]，说明 OpenClaw 已跨越"好用的工具"到"值得投身的社区"的门槛。
