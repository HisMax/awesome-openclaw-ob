# Cursor 分析

## 基本信息

- **类别**：AI 原生 IDE，属于 [[Agentic Coding]] 范式
- **定位**：代码编辑、内联补全、多文件重构
- **核心功能**：完整的编辑器体验（Composer）
- **模型灵活性**：受限于集成模型（多模型，包括 Claude 和 GPT）
- **上手难度**：下载安装 5 分钟
- **许可**：商业

## 定价

- **Pro**：$20/月

## 市场表现

- ~**18%** 市场份额（付费 AI 编码工具市场）
- **200 万+** 用户，**100 万+** 付费，**5 万+** 企业团队
- ~~$10 亿+ ARR（2025 年底）~~ → **$30 亿 ARR**（2026.4），史上最快从 $0 → $20 亿 ARR 的 SaaS（参见 [[AI 编码助手市场数据]]）
- 60% 收入来自企业客户，近 70% Fortune 1000 公司使用

## 隐私考量

代码经 Cursor 服务器处理，与 OpenClaw 的本地运行模式形成对比。

## 适用场景

在行业最佳实践中，Cursor 定位为 **IDE 内写代码** 的首选工具。

## 最新动态（2026年3月）

- **Automations 功能发布**（3月5日）：Cursor 推出了"始终在线"自主编程系统 [[Cursor Automations]]，支持事件驱动触发（代码提交、Slack 消息、PagerDuty 告警）和多智能体并行。这是 Cursor 从"AI 辅助 IDE"向"自主编码平台"的关键一步
- Automations 让 Cursor 在功能上直接对标 Devin，但价格仍然只有 $20/月
- 在 Anthropic Agentic Coding 趋势报告中，Cursor 被引用为"开发者工作流根本重构"趋势的代表产品
- 详见 [[Cursor Automations]] 的完整分析
