## 编码工具竞争

传统 [[Agentic Coding|AI 编码工具]]的界面差异：
- Claude Code -> 终端（命令行）
- Cursor -> IDE（开发环境）
- Devin -> 专业工程 UI（面向工程团队）
- **[[OpenClaw 是什么|OpenClaw]]** -> WhatsApp/Telegram/Discord（面向所有人）

OpenClaw 的差异化在于编程民主化——用聊天界面服务非技术用户，实现可及性突破。其 [[模型无关架构]] 和开源策略是区别于大厂产品的核心优势。

## 竞争加剧的风险

Microsoft Copilot Tasks、Claude Cowork 等大厂产品入场是投资风险之一。这也是 [[大厂 Agent 竞赛]] 的缩影。
