# Replit Agent 分析

## 基本信息

- **类别**：AI 编码代理
- **定位**："[[Vibe Coding|vibe coding]]" 快速原型，编码民主化
- **目标用户**：非技术人员 → 软件构建者（"十亿开发者"愿景）

## 市场表现

- ~~$30 亿估值（2025.9）~~ → **$90 亿估值**（2026.3，$4 亿 Series D 融资）
- 从 $280 万飙升至 **$1.5 亿年化收入**，目标 2026 年底达 **$10 亿 ARR**
- **75 万+** 企业用户

## 旗舰案例

**Rokt**：24 小时内构建 135 个内部应用（700+ 员工参与）

## Agent 4（2026.3 发布）

- **Design Canvas**：取代旧 Design Mode，无限画板支持所有 artifact 类型和直接操作
- **协作模式重构**：取消 fork-and-merge，改为共享项目 + Agent 辅助合并（90% 冲突自动解决）
- **Plan-while-building**：不再需要先规划后构建，两个阶段可并行推进
- **多格式支持**：slides、网站、Web App、移动 App 均可一句话创建
- **并行任务**：支持多 Artifact 并行构建

> Agent 3 能力保留：自主性提升 10 倍，可连续工作 200 分钟

## 与 OpenClaw 的差异

两者的"降低门槛"方向不同，但都依赖 LLM 驱动：
- **Replit** 降低的是**编码门槛**（让非技术人员也能构建应用），属于 [[Agentic Coding]] 范畴
- **[[OpenClaw 是什么|OpenClaw]]** 降低的是**AI 代理的使用门槛**（通过聊天界面自动化日常生活）
