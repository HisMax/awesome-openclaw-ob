# OpenClaw vs Cursor

| 维度 | [[OpenClaw 是什么|OpenClaw]] | [[Cursor 分析|Cursor]] |
|------|----------|--------|
| **类别** | 自主 AI 代理 | AI 原生 IDE |
| **核心** | 跨应用任务自动化 | 代码编辑、内联补全、多文件重构 |
| **隐私** | 本地运行，代码不离开本机 | 代码经 Cursor 服务器处理 |
| **模型灵活性** | 随时切换任何模型（Claude、GPT、DeepSeek 等） | 受限于集成模型 |
| **IDE 集成** | 无 IDE 集成、无内联补全 | 完整的编辑器体验（Composer） |
| **上手难度** | Docker/SSH/终端，30-45 分钟 | 下载安装 5 分钟 |
| **定价** | 免费 + API | $20/月 Pro |

## 结论

**互补工具而非竞品**。Cursor 用于 IDE 内写代码，OpenClaw 用于 IDE 外的系统自动化。
