# 竞品成本对比

## Claude Code 定价（2026.2）

| 计划 | 月费 | 用量 | 适合人群 |
|------|------|------|----------|
| **Pro** | $20/月 | 基础用量，含 Claude Code | 轻度使用者 |
| **Max 5x** | $100/月 | 5 倍 Pro 用量（~225+ 条/5h） | 日常专业开发者 |
| **Max 20x** | $200/月 | 20 倍 Pro 用量（~900+ 条/5h） | 重度用户，全天使用 |

> 实际案例：一位开发者 201 个会话、45+ 项目的 API 等价成本为 **$5,623**——超过 Max 5x 计划 5 年费用。

## OpenClaw 定价（2026.2）

| 配置 | 月费 | 说明 |
|------|------|------|
| **纯本地** | **$0** | Ollama 本地模型，零 API 成本，性能受限 |
| **经济配置** | **$13-18/月** | DeepSeek/小模型 API，日常够用 |
| **中端配置** | **$5-15/月** | Claude Sonnet / GPT-4o-mini 按量计费 |
| **极端使用** | **$623/月** | 持续高频调用 Opus/GPT-4 级模型（月成本实录） |
