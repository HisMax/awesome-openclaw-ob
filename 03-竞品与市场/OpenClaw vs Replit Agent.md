# OpenClaw vs Replit Agent

| 维度 | [[OpenClaw 是什么|OpenClaw]] | [[Replit Agent 分析|Replit Agent]] |
|------|----------|--------------|
| **估值/融资** | 开源项目（MIT） | **$90 亿估值**（2026.3，$4 亿 Series D） |
| **年收入** | $0（开源） | $1.5 亿年化收入，目标 2026 年底 **$10 亿 ARR** |
| **代理能力** | 24/7 消息助手 | Agent 4 发布（2026.3）：Design Canvas + 并行任务 + plan-while-building |
| **旗舰案例** | 社区驱动，个人自动化 | **Rokt：24 小时内构建 135 个内部应用**（700+ 员工参与） |
| **目标用户** | 技术爱好者 / 极客 | 非技术人员 → 软件构建者（"十亿开发者"愿景） |
| **定位** | 通用 AI 助手（消息集成 + 持久运行） | AI 编码代理（"vibe coding" 快速原型） |

## 关键差异

两者的"降低门槛"方向不同：
- **Replit** 降低的是**编码门槛**（让非技术人员也能构建应用）
- **OpenClaw** 降低的是**AI 代理的使用门槛**（通过聊天界面自动化日常生活）
