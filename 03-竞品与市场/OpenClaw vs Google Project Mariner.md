# OpenClaw vs Google Project Mariner

| 维度 | [[OpenClaw 是什么|OpenClaw]] | [[Google Project Mariner 分析|Google Project Mariner]] |
|------|----------|------------------------|
| **类别** | 通用个人 AI 代理 | 浏览器自动化 AI 代理 |
| **架构** | 本地守护进程 + 消息桥接 | 云端 VM + Chrome 扩展 |
| **并发** | 单实例单任务 | 最多 10 个并发任务 |
| **基准** | 无标准化基准 | WebVoyager 基准 **83.5% 成功率**（SOTA，单代理） |
| **模型** | 多模型 | Gemini 2.0 |
| **限制** | 安全风险依赖用户配置 | 不能完成购买、接受 Cookie、同意服务条款 |
| **定位** | 跨应用通用个人助手 | 专注浏览器任务自动化 |

## 关键差异

Mariner 是浏览器内的自动化专家（83.5% 任务成功率），但竞品 Browserable 已达 90.4%、Magnitude 94%。

OpenClaw 是消息界面中的通用个人助手，不局限于浏览器场景。两者都是 Agentic AI 浪潮中的重要产品，但面向的自主执行领域截然不同。

## 2026年Q2更新

> **Google 于 2026.5.4 正式关停 Project Mariner**。核心技术被并入 Gemini Agent 和 Chrome auto-browse 功能。本对比文件仅作为历史参考保留。
