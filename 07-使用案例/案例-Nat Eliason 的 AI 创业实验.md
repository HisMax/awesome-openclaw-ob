# 案例：Nat Eliason 的 AI 创业实验


![[assets/nat-eliason.jpg]]

**人物**：Nat Eliason，知名创作者经济博主
**案例**：给 OpenClaw 机器人 "Felix" $1,000 启动资金，让它自主创业

## 成果

- 3 周内赚取 **$14,718**
- 自主启动了网站、信息产品和 X 账号
- 目前运营速度：**$4,000/周**

## 核心创新：三层记忆系统

这套 [[三层记忆系统]] 是该案例的技术亮点：

1. **知识图谱层**：PARA 系统（Projects/Areas/Resources/Archives）在 `~/life/` 目录
2. **每日笔记层**：日期标注的 Markdown 文件，夜间自动整合到知识图谱
3. **隐性知识层**：存储沟通偏好、工作流习惯、硬性规则

## 启示

此案例展示了 Agentic Coding 在商业领域的潜力——Agent 不仅能写代码，还能自主经营业务。与 [[案例-ClawWork 经济基准]] 的学术测试不同，这是一个真实的商业实验。Felix 的成功也验证了 Tool Use 机制 在跨领域任务（网站构建、内容创作、社交媒体运营）中的通用性。

## 技术与商业洞察

Felix 的[[记忆系统]]设计（三层架构）是它持续学习和自主决策的基础。通过 Agent Execution Loop 不断积累经验，Agent 的表现随时间改善。$1,000 到 $14,718 的增长验证了 [[编程民主化]] 和 可及性突破 的商业价值——不需要创始团队，一个配置得当的 Agent 就能启动一项业务。这个案例也为 OpenClaw 商业模式 和 商业化路径 提供了真实的收入验证数据。大语言模型驱动的 Agent 在选择商业策略时展现的判断力，超出了传统提示工程所能达到的范围。
