# 案例：ClawWork 经济基准

**一句话总结**：香港大学团队构建了第一个系统性衡量 AI Agent "赚钱能力"的基准测试——Agent 从 $10 启动资金出发，7 小时内赚了 $10,000，覆盖 44 个行业的 220 个真实任务，为"AI 劳动力"的经济价值提供了学术级量化证据。

**项目**：ClawWork（香港大学团队）
**案例**：将 AI Agent 变成"经济上负责"的 AI 同事

## 关键数据

- Agent 从 **$10 启动资金**开始
- 完成 **220 个真实职业任务**（覆盖 44 个行业）
- 7 小时内赚取 **$10,000**
- 顶尖模型达到等价 **$1,500+/小时** 的工作效率
- 任务价值范围：$82.78 – $5,004

## 研究设计

### 为什么需要经济基准？

在 ClawWork 之前，衡量 AI Agent 能力的基准主要是技术性的——SWE-bench 测代码能力（Claude Opus 4.6 达 80.8%）、WebVoyager 测浏览器操作（Project Mariner 达 83.5%）。但没有一个基准回答这个最实际的问题：**AI Agent 能赚多少钱？**

ClawWork 填补了这个空白。它的创新在于：
- **真实任务**：不是模拟场景，而是来自真实自由职业市场的真实付费任务
- **跨行业覆盖**：44 个行业意味着结果不局限于编程，也包括文案、设计、数据分析、市场研究等
- **经济指标**：用"时薪"和"净利润"而非"准确率"来衡量——这对企业决策者更有意义

### 任务类型分布

220 个任务覆盖了广泛的职业领域：
- **技术任务**：代码编写、数据库设计、自动化脚本
- **内容创作**：文案写作、社交媒体内容、SEO 优化
- **数据分析**：财务报表分析、市场研究、数据可视化
- **行政任务**：邮件处理、日程管理、文档整理
- **专业服务**：法律文件初稿、会计记录、技术文档

## 核心洞察

### $1,500/小时意味着什么？

这个数字需要放在上下文中理解：
- **美国律师**平均时薪 $200-500
- **高级管理咨询师**时薪 $300-600
- **华尔街分析师**等价时薪 $400-800
- **ClawWork 顶尖 Agent**：$1,500+/小时

当然，这个数字有重要的注意事项：它衡量的是 Agent 在理想条件下的峰值效率，不代表持续稳定的产出。但即使打个对折，也远超大多数人类知识工作者的效率。

### 学术基准 vs 真实世界

ClawWork 与 [[案例-Nat Eliason 的 AI 创业实验]] 的真实商业实验形成互补：

| 维度 | ClawWork | Nat Eliason 实验 |
|------|----------|-----------------|
| **性质** | 学术基准测试 | 真实商业实验 |
| **时间** | 7 小时 | 3 周 |
| **收入** | $10,000 | $14,718 |
| **起始资金** | $10 | $1,000 |
| **可重复性** | 高（标准化任务） | 低（依赖具体市场条件） |
| **覆盖面** | 44 行业、220 任务 | 单一商业项目 |

两者合在一起提供了一个更完整的图景：**AI Agent 的经济价值是真实的，无论在实验室还是真实市场中都得到了验证。**

### 对劳动力市场的影响

ClawWork 的数据揭示了一个深刻的趋势：

1. **AI 劳动力的成本优势**：Agent 完成一个 $500 的任务，API 成本可能只有 $5-20——这是 25-100 倍的投入产出比
2. **速度优势**：7 小时完成 220 个任务，平均每个任务不到 2 分钟。人类完成同样的任务可能需要数天
3. **规模化能力**：一个 Agent 可以被复制无限次，同时处理大量并行任务

这些数据支撑了 [[半人马阶段与 AI Agent 狂潮]] 中 James Wang 的观察——使用 AI 工具的人和不使用的人之间的鸿沟正在急速扩大。如果一个 Agent 能以 $1,500/小时的效率工作，那么雇佣它的人和没有使用它的竞争对手之间的差距将是指数级的。这正是 [[编程民主化]] 和 可及性突破 所带来的深远影响。

## 与 OpenClaw 生态的关系

ClawWork 虽然是独立的学术项目，但它使用的 Agent 架构与 [[OpenClaw 是什么|OpenClaw]] 类工具高度相关：
- Agent 需要访问浏览器、API、文件系统（正是 OpenClaw 的 [[Tool Use 机制]] 提供的能力）
- 多模型路由（选择最适合任务的模型）是 OpenClaw 模型无关架构的核心特性之一
- 持久记忆和跨任务学习与 OpenClaw 的跨会话持久记忆一脉相承
- 经济基准的结果也为 [[API 定价与成本分析]] 提供了重要的参考框架——ClawWork 的经济基准证明 Agent 的实际成本远低于人工，一个 $500 任务的 API 成本仅 $5-20
