# 案例：Fast Company 记者亲测


![[assets/fast-company.jpg]]

**一句话总结**：一位两届 ASME 获奖的资深科技记者用 OpenClaw 训练 AI 做他的本职工作——写稿，结果"令人惊讶也有点吓人"，但更重要的发现是：大厂畏缩不前，而开源社区率先让普通人体验到了真正的 AI Agent 自主性。

**人物**：Mark Sullivan，Fast Company 资深撰稿人，AI Decoded 周刊作者，两届 ASME（美国杂志编辑协会）获奖记者
**案例**：训练 OpenClaw 做他的本职工作——为 Fast Company 撰写稿件

## 实验背景

Mark Sullivan 不是一个随便的博主——他是专业科技记者，在 Fast Company 拥有固定专栏 "AI Decoded"。他选择让 AI 做的不是简单任务，而是他赖以为生的核心技能：新闻写作。这让这个实验具有了特殊的说服力和紧张感。

他的目标是：看看 OpenClaw 能否完成从选题研究、信息搜集、到初稿撰写的完整新闻写作流程。

## 核心发现

- 将 OpenClaw 描述为 "kludgey, wildly complex, open-source platform"——粗糙、极度复杂的开源平台
- 结果被描述为**"令人惊讶——也有点吓人"（surprising and scary）**
- 大型 AI 公司承诺了 Agentic AI 未来，但"过于畏缩和成本敏感"而不敢真正发布
- OpenClaw 成为第一个让普通人真正体验 AI Agent 自主性的工具

"令人惊讶"在于 Agent 确实能产出接近可发表水平的内容；"吓人"在于这对记者职业的潜在威胁——如果一个开源工具就能做到这些，那么专业投入数十亿美元的 AI 公司最终会做出什么？

## 关于"大厂畏缩"的洞察

Sullivan 的观察切中了 2026 年初 AI 行业的一个核心矛盾：

- **OpenAI、Google、Anthropic** 都在内部拥有远超 OpenClaw 的 Agent 技术
- 但出于**品牌风险、法律责任、安全顾虑**，它们选择了缓慢释放能力
- OpenClaw 作为开源项目，没有这些顾虑——"move fast and break things" 的精神在这里重现
- 结果是：**一个粗糙的开源工具反而成为公众第一次真正体验 AI Agent 自主性的入口**

这解释了为什么 TechCrunch 引用的 AI 研究者会说 "from an AI research perspective, this is nothing novel"——技术本身不新，但**敢于发布并让普通人使用**才是 OpenClaw 的真正创新。

## 记者视角的独特价值

作为非技术人员的专业评测，Sullivan 的体验揭示了几个关键点：

1. **上手门槛极高**：即使是科技记者也觉得 OpenClaw "wildly complex"——这印证了为什么 Claude Cowork 选择了 macOS GUI 作为界面，瞄准"非技术知识工作者"
2. **结果令人印象深刻**：尽管过程痛苦，最终产出超出预期——说明 Agent 的价值主张是真实的，只是 UX 还没跟上
3. **职业威胁感是真实的**：一个获奖记者说"有点吓人"，这比任何技术基准测试都更能说明 AI Agent 的进展

## 核心洞察

Sullivan 的实验可以看作 [[半人马阶段与 AI Agent 狂潮]] 的一个具体缩影：

- **半人马模式成立**：记者 + AI Agent 的组合能力超过纯记者或纯 AI
- **但窗口在收窄**：如果 Agent 已经能写出"surprising"质量的稿件，那么下一代模型呢？
- **UX 是最后的护城河**：OpenClaw 的"kludgey"体验可能是唯一阻止它取代更多知识工作者的因素

## 与其他案例的关系

与 [[案例-Reorx 从程序员到 Tech Lead]] 的开发者视角互为补充——前者是技术人员发现自己角色变了（从程序员到 Tech Lead），后者是非技术人员发现自己能做以前做不到的事。

与 [[案例-Ken Yeung 的 Vibe Coding]] 共同印证了 [[编程民主化]] 的趋势：Ken 是非技术人员构建了产品，Sullivan 是非技术人员让 AI 做了专业工作。两者都说明 AI Agent 正在重新定义"谁能做什么"的边界。

同时，这也是主流媒体对 OpenClaw 最深度的亲身体验报道之一（Lex Fridman 的 2 小时播客侧重创始人故事，Sullivan 的评测侧重用户体验），为理解 OpenClaw 的真实能力边界提供了珍贵的第一手素材。这一案例也佐证了 Vibe Coding 和可及性突破的核心论点：大语言模型驱动的 Agent 正在让专业能力的边界变得模糊。
