# Sam Altman

> **一句话总结**：OpenAI CEO，宣称 2025 为 Agent 元年，收编 OpenClaw 创始人 Steinberger，推动 AI Agent 从实验走向商业化核心战略。

## 核心要点

- 2025.1 宣称"2025 年可能出现首批加入劳动力市场的 AI Agent"，同时声称 OpenAI"已知如何构建 AGI"
- 2026.2.14 收编 [[Peter Steinberger]]，公开称其为"genius"，将 Agent 战略提升为核心产品方向
- 与 [[Mark Zuckerberg 与 OpenClaw|Zuckerberg]] 竞争争夺 Steinberger，反映大厂对个人 Agent 赛道的战略重视
- 推动 OpenAI 联合创立 [[AAIF 基金会]]，布局 Agent 标准化

## 详细内容

### "Agent 元年"宣言（2025.1，《Reflections》）

> "We believe that, in 2025, we may see **the first AI agents join the workforce** and materially change the output of companies."

这篇博文同时宣称 OpenAI"已经知道如何构建 AGI"，将公司战略从聊天机器人转向超级智能和 Agent 生态。但现实数据显示差距：仅 **11%** 的组织在生产环境部署了 Agent（Deloitte），**35%** 甚至没有 Agent 策略。

### 收编 Steinberger（2026.2.14-15）

Altman 宣称 Agent 元年，然后立刻行动收编了最火的开源 Agent——详见 [[OpenAI 为什么收编 OpenClaw]]。

Altman 在 X/Twitter 上的背书推文：

> "Peter Steinberger is joining OpenAI to drive the next generation of personal agents. He is a genius with a lot of amazing ideas about the future of very smart agents interacting with each other to do very useful things for people. **We expect this will quickly become core to our product offerings.**"

该推文获得 **450 万浏览、3.9K 回复、5.4K 转发、37K 点赞**。

### 收编背后的战略逻辑

| 因素 | 说明 |
|------|------|
| 用户入口 | [[OpenClaw]] 是最成功的"AI Agent → 普通用户"桥梁 |
| 模型消耗 | 237K stars 项目意味着巨大的 API 调用量 |
| 生态卡位 | 5,705 个技能的生态系统 = 开发者粘性 |
| 竞争防御 | 防止被 Google/Meta 获取成为对手的分发渠道 |

### 多 Agent 愿景

> "**The future is going to be extremely multi-agent** and it's important to us to support open source as part of that."

关键条件：OpenClaw 保持开源和中立，OpenAI 赞助基金会运营，项目继续支持多模型。Steinberger 获得六位数以上 offer。

### 与 Karpathy 的分歧

| | Sam Altman | [[Andrej Karpathy]] |
|---|---|---|
| Agent 时间线 | 2025 年即入劳动力市场 | 需要 5-10X 更长时间 |
| 态度 | 激进乐观 | 悲观中的乐观 |
| 证据 | 战略愿景驱动 | 技术限制驱动（APEX-Agents < 25%） |

### GPT-5.3 Codex 与模型竞争

2026.2.5 发布的 GPT-5.3 Codex 在 SWE-Bench Verified 达 **80.0%**，Terminal-Bench **77.3%**，与 [[Dario Amodei|Anthropic]] 的 Opus 4.6（80.8%）差距不到 1 个百分点。16 天内三大前沿模型同时发布，Agent 编码能力进入"平台期竞争"。

### GPT-5.5 发布与 Codex 全面开放（2026.4）

2026.4.23，OpenAI 正式发布 **GPT-5.5**，面向 Plus、Pro、Business、Enterprise 全部付费用户开放。联合创始人 Greg Brockman 称其为构建"super app"的又一步——一个瑞士军刀式的多功能应用。GPT-5.5 定位为"高度独立的数字工作者"，能自主浏览网页、编写代码、操作不同软件工具来完成项目。

Altman 在推特上称 GPT-5.5 为"an autistic genius with very strange taste in naming"，并对其自主完成 Codex 任务的能力赞赏有加。

Codex 的付费用户已突破 **200 万周活跃用户**，Business/Enterprise 用户数自 1 月以来增长了 **6 倍**。2026.4.2 起 Codex 定价从按消息计费改为与 API token 用量对齐，覆盖所有付费计划。ChatGPT 付费商业用户已超 **900 万**。

关于下一代模型，Polymarket 预测 GPT-5.6 有 68% 的概率在 2026 年 6 月 8-14 日发布。

### Steinberger 的 API 消耗（2026.5）

Steinberger 加入 OpenAI 后，展示了一张 30 天 API 账单截图：**$1,305,088.81**——603 亿 tokens，760 万次请求，全部由约 100 个 Codex 实例和 3 人团队产生。这是 Altman "multi-agent" 愿景的极端实践。

## 关键洞察

Altman 的 Agent 元年论断在 2025 年看起来过于激进（Deloitte 数据显示仅 11% 落地），但 2026 年 [[OpenClaw]] 的爆发某种程度上验证了方向的正确性——只是时间线推迟了一年。他收编 Steinberger 的决策同时是攻击和防御：既获取了最热门 Agent 项目的人才和生态影响力，又防止了 [[Mark Zuckerberg 与 OpenClaw|Meta]] 或其他竞争对手获取这一资源。[[Dario Amodei]] 的商标执法间接将 OpenClaw 推向了 OpenAI 阵营，Altman 精准地抓住了这个窗口。
