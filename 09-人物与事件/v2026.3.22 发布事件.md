# v2026.3.22 发布事件

## 一句话理解

> 2026 年 3 月 23 日，OpenClaw 发布了有史以来规模最大的版本 v2026.3.22——100+ 贡献者、22 项 Breaking Changes、全面的架构重构，并在同日紧急跟进了 v2026.3.23 热修复。这一天标志着 OpenClaw 从"创始人驱动"正式进入"社区驱动"时代。

## 时间线

| 时间 | 事件 |
|------|------|
| 3月23日（早） | [[Peter Steinberger\|@steipete]] 签名发布 v2026.3.22-beta.1 |
| 3月23日（中） | v2026.3.22 正式版发布，241 commits since last release |
| 3月23日（晚） | v2026.3.23 热修复紧急发布，修复捆绑插件 sidecar 等关键问题 |

## 核心影响

### 架构层面
- [[Provider-Plugin 架构]]：捆绑提供商（OpenRouter/Copilot/Codex）完成插件化迁移
- [[可插拔沙箱后端]]：引入 [[OpenShell]] 和 SSH 后端
- [[Plugin 扩展系统]]：SDK 路径从 `openclaw/extension-api` 迁移到 `openclaw/plugin-sdk/*`
- [[ClawHub 原生安装]]：Skills 安装流程整合进核心

### 生态层面
- 7+ 新模型提供商：[[Anthropic Vertex 提供商|Anthropic Vertex]]、[[Xiaomi MiMo 提供商|MiMo]]、[[Qwen DashScope 提供商|Qwen]]、[[Z.AI GLM 4.5-4.6 系列|GLM]]、[[xAI Grok 集成|Grok]]、[[MiniMax M2.7 系列|MiniMax]]
- 4 个新搜索/爬取插件：[[Exa 网页搜索插件|Exa]]、[[Tavily AI 搜索插件|Tavily]]、[[Firecrawl 爬取插件|Firecrawl]]、[[Chutes 推理加速插件|Chutes]]
- [[飞书集成]]：深度企业协作平台集成

### 安全层面
- [[沙箱运行时注入防护]]：阻断 JVM/Glibc/.NET 注入
- [[SSRF 固定与代理安全]]：网络层安全加固
- [[macOS 执行白名单加固]]：执行层安全加固
- [[Marketplace 安装范围验证]]：供应链安全

### 社区层面
- [[v2026.3.22 社区贡献|100+ 贡献者]]参与
- v2026.3.23 收到 173 个社区反应
- [[OpenClaw GitHub 数据更新 2026Q1|Stars 达到 334K]]

## 历史定位

如果说 v2.1 是 OpenClaw 的"安全补课"（回应 [[ClawHavoc 事件]]），v2026.3.12 是"模块化开端"，那么 v2026.3.22 就是**"平台化宣言"**——这个版本同时在插件 SDK、沙箱后端、提供商架构、安全基线四条线上完成重构，清晰宣告 OpenClaw 正在从"功能框架"变为"可组装平台"。

但同日热修复的发布也提醒我们：如此大规模的架构重构仍有稳定性债务需要偿还。
