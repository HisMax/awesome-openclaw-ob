# The Great Molt 改名事件


![[assets/great-molt.jpg]]

> **一句话总结**：Clawdbot 因 Anthropic 商标通知被迫改名为 OpenClaw 的 3 小时全平台紧急迁移事件，过程中伴随加密骗局、账号抢注、GitHub 事故和社区通宵抢救。

## 核心要点

- 完整名称演变链：**Warelay → Clawd → Clawdbot → Moltbot → OpenClaw**
- [[Dario Amodei|Anthropic]] 2026.1.27 发出商标通知，"Clawd"与"Claude"过于相似
- 改名间隙爆发 [[$CLAWD 加密骗局]]，市值飙至 $1600 万后崩盘
- 最终迁移在 **3 小时内**完成，社区核心成员通宵协作

## 详细内容

### 完整时间线

| 时间 | 事件 |
|------|------|
| 2025.11（原型） | 最初名为 **Warelay**（WhatsApp Relay 缩写） |
| 2025.11（公开） | 更名为 **Clawd / Clawdbot** 公开发布 |
| 2026.1.27 | [[Dario Amodei|Anthropic]] 发出商标通知 |
| 2026.1.27 | [[Peter Steinberger]] 询问"Clawbot"是否可行，被告知"Not allowed to" |
| 改名间隙 | [[$CLAWD 加密骗局]]爆发 |
| 2026.1.27-29 | 社区凌晨 5 点 Discord 投票更名为 **Moltbot**（龙虾蜕壳之意） |
| 2026.1.30 | 因发音困难最终定名 **OpenClaw** |

### 混乱的改名过程

改名过程堪称灾难性，暴露了互联网身份迁移的脆弱性：

**账号抢注风暴**
- Twitter 改名后**几秒内**机器人抢注 @openclaw 发布加密钱包骗局
- 改名间隙旧社交账号被诈骗团伙在 **10 秒内**抢注

**GitHub 事故**
- Steinberger 意外改了自己的 **GitHub 个人账户名**而非组织名
- **steipete** 用户名被立即抢注
- 需要 **GitHub SVP** 介入才得以恢复

**"Handsome Molty"恐怖事件**
- 尝试让 [[龙虾吉祥物文化|吉祥物 Molty]] "老 5 岁"的 AI 生成请求
- 意外产生了**恐怖的人脸龙虾体图像**
- 加密骗子立即将其武器化为 meme 进行诈骗宣传

### Steinberger 的情感冲击

在 Lex Fridman 播客中承认：

> "I was close to crying"

差点删除整个项目。睡了一觉后，以"**曼哈顿计划**"级别的保密性策划了第二次改名——使用诱饵名称、跨平台同步更换、提前致电 [[Sam Altman]] 确认新名称不侵权。

### 社区的文化重构

改名风波被社区重新诠释为积极叙事——不是毁灭而是"exfoliate"（脱壳），脱去旧壳以更强大地成长。"Molt"成为 [[龙虾吉祥物文化]] 的核心隐喻：

- **蜕壳 = 成长**：龙虾通过蜕壳才能长大
- **社区韧性**：危机中的通宵协作证明了社区的真实凝聚力
- **5 天内募集 $35,000**：项目转交基金会后的社区捐款

### 外界反应

- [[DHH 对 Anthropic 的批评|DHH（Ruby on Rails 创建者）]]：公开批评 Anthropic 为"customer hostile"
- 讽刺核心：OpenClaw 是 Anthropic Claude API **最大的付费流量来源之一**
- 最终结果：商标执法间接将 Steinberger 推向了 [[Sam Altman|OpenAI]]

## 关键洞察

The Great Molt 事件是互联网时代**数字身份脆弱性**的极端案例。一个拥有 237K GitHub stars 的项目，其品牌迁移可以在数秒内被机器人劫持、被加密骗子武器化。事件也展示了开源社区的两面性：一方面社区凝聚力惊人（3 小时全平台迁移），另一方面社区的开放性本身就是攻击面。[[Dario Amodei|Anthropic]] 的商标执法在法律上正当，但其连锁效应（[[$CLAWD 加密骗局]]、创始人流失、社区信任危机）远超预期，成为科技行业品牌保护与开源生态冲突的经典案例。
