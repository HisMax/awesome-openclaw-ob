# 龙虾吉祥物文化

[[OpenClaw 是什么|OpenClaw]] 的龙虾吉祥物 **Molty** 已成为项目最鲜明的文化符号。

## Molty 的身份

- 使用 they/them 代词
- 代表通过 markdown [[记忆系统|记忆文件]]维持持久身份的 AI 实例——不是"他"或"她"，而是一个有记忆、有性格的 Agent 人格

## 蜕壳隐喻

[[OpenClaw 的起源与发展历程|命名风波]]被社区重新诠释为积极叙事——不是毁灭而是 "exfoliate"（脱壳），脱去旧壳以更强大地成长。

## 社区传统用语

- **"Yee-claw"**：社区打招呼方式（牛仔风龙虾致意）
- **"Claw abiding citizens"**：社区成员自称
- **"Clawntroversy"**：用于讨论项目争议话题
- **"The claw is the law"**：核心口号，强调社区自治和开源精神

## Moltiverse 与社区活动（2026 年中更新）

龙虾文化已从线上 meme 扩展为线下社区运动：

- **Moltiverse**：社区为围绕 OpenClaw 的整个生态起了一个专属名词——"Moltiverse"（蜕壳宇宙），AI Agent 在其中"molt, grow, and evolve"
- **旧金山首次线下 Meetup**：吸引了大量参与者，现场充满"this is the future"的氛围，重点讨论安全改进
- **龙虾的象征意义**被进一步诠释：龙虾通过蜕壳成长，而 OpenClaw 从 Warelay → Clawd → Moltbot → OpenClaw 的四次蜕变，完美体现了这一隐喻
- 社区规模从早期 Discord 核心圈扩展至 **320 万月活用户**（2026.4），GitHub 贡献者超 **1,200 人**
