# v2026.3.22 Breaking Changes 迁移指南

## 一句话理解

> [[OpenClaw v2026.3 版本更新|v2026.3.22]] 引入了 OpenClaw 历史上最大规模的 Breaking Changes，涉及插件 SDK 路径、环境变量、浏览器集成、模型配置和安全运行时的全面重构。

## 影响范围

v2026.3.22 的 Breaking Changes 影响面极广：

- [[ClawHub 官方技能注册表]] 上的 3,286 个 Skills 中依赖旧 SDK 路径的需要迁移
- 所有使用 `openclaw/extension-api` 的第三方插件
- 使用 Chrome 扩展中继方式的浏览器控制配置
- 依赖 `.moltbot` 状态目录或 `CLAWDBOT_*`/`MOLTBOT_*` 环境变量的部署
- 使用 `nano-banana-pro` 图像生成 Skill 的用户

## 迁移清单

### 1. 插件 SDK 路径迁移

**变更**：`openclaw/extension-api` → `openclaw/plugin-sdk/*` 子路径

```diff
- import { Tool } from 'openclaw/extension-api'
+ import { Tool } from 'openclaw/plugin-sdk/core'
```

捆绑插件不再直接 import 宿主模块，改为通过注入的运行时操作。消息工具发现必须使用 `ChannelMessageActionAdapter.describeMessageTool(...)`。

### 2. 环境变量清理

**变更**：移除所有 `CLAWDBOT_*` 和 `MOLTBOT_*` 环境变量，统一为 `OPENCLAW_*`

这是 [[OpenClaw 的起源与发展历程|改名事件]] 的最终技术清理。搜索代码和配置中的旧前缀并替换。

### 3. 浏览器配置迁移

**变更**：Chrome MCP 扩展中继路径移除

运行 `openclaw doctor --fix` 自动将本地浏览器配置迁移到 `existing-session`/`user` 模式。`driver: "extension"` 和 `browser.relayBindHost` 配置项弃用。

### 4. 模型配置更新

- 默认 OpenAI 模型切换为 `openai/gpt-5.4`
- 图像生成统一为 `image_generate` 工具，使用 `agents.defaults.imageGenerationModel` 配置
- `nano-banana-pro` Skill 包装移除

### 5. Matrix 插件迁移

全新 Matrix 插件基于官方 `matrix-js-sdk`，需按官方迁移指南操作。

### 6. ClawHub 安装优先级变更

`openclaw plugins install <package>` 对 npm 安全名称优先查询 ClawHub。如需强制 npm 安装，需显式指定来源。

## 安全相关 Breaking Changes

- 沙箱阻止 JVM 注入环境变量（`MAVEN_OPTS`/`SBT_OPTS`/`GRADLE_OPTS`/`ANT_OPTS`）
- 阻止 `GLIBC_TUNABLES` 和 `DOTNET_ADDITIONAL_DEPS`
- Discord 命令部署切换为 Carbon reconcile
- 语音通话 Webhook 预认证限制

## 自动修复工具

```bash
openclaw doctor --fix
```

此命令可自动处理部分迁移，包括浏览器配置迁移和基础配置修复。
