# OpenClaw 与 Anthropic 关系

OpenClaw 与 [[Anthropic 公司分析|Anthropic]] 之间存在微妙的竞争动态。

## 关键事实

- OpenClaw 最初名为 **Clawdbot**（Claude 的双关语），详见 [[OpenClaw 的起源与发展历程|起源历程]]
- Anthropic 发出**商标通知**迫使改名为 OpenClaw
- OpenClaw 采用 [[模型无关架构]]，支持 Claude 模型但**不限于** Claude，还支持 GPT、DeepSeek 等
- 创始人 [[Peter Steinberger]] 最终加入了 **OpenAI** 而非 Anthropic
- Claude Code 是 OpenClaw 在编码领域的直接竞品（见 [[OpenClaw vs Claude Code]]）

## 竞争启示

Anthropic 的商标行动可能间接将一个强大的开源生态推向了 OpenAI 阵营。这是一个值得注意的战略失误案例。

不过，Anthropic 并未因此受损——[[Anthropic Series H 融资|Series H 后估值 $9,650 亿]]，ARR $470 亿，Claude Code 年化营收超 $25 亿。Anthropic 用商业成功证明了即使失去 OpenClaw 生态的直接控制，自身 Agent 产品线依然强劲。
