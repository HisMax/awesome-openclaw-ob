# Deloitte 企业 AI Agent 采用报告

## 一句话总结

Deloitte 2025/2026 调查显示，仅 11% 的组织在生产环境部署了 AI Agent，38% 在试点阶段，42% 仍在制定策略，揭示了 Agent 技术的炒作与现实之间的巨大落差。

## 核心数据

| 指标 | 比例 |
|------|------|
| 生产环境部署 AI Agent | **11%**（早期调查） |
| 正在试点 Agent | **38%** |
| 仍在制定 Agent 策略 | **42%** |
| 完全没有 Agent 策略 | **35%** |
| 数据来源 | Deloitte 2025/2026 调查 |
| 相关报告 | Deloitte Tech Trends 2026 |

### 2026 年更新数据（State of AI in the Enterprise 2026）

| 指标 | 比例 |
|------|------|
| 已至少中等程度使用 Agentic AI | **23%** |
| 预计两年内达到中等以上使用 | **74%** |
| 计划定制 Agent 以适应业务需求 | **85%** |
| 拥有成熟的 AI Agent 治理模型 | 仅 **20%** |
| 从 AI 获得生产力和效率提升 | **66%** |
| 用 AI"深度转型"业务 | 仅 **34%** |
| 调查样本 | 3,235 名业务和 IT 领导者，24 国 6 行业 |

## 分析

Deloitte 的两轮数据清晰地描绘了 Agent 采用的加速曲线。早期调查的 11% 生产部署率在 2026 年更新报告中升至 23%——增长了一倍多，印证了市场正从试点走向规模化。

但治理缺口更加突出：仅 20% 的企业拥有成熟的 AI Agent 治理模型，而 85% 计划定制 Agent。这意味着**大多数企业在没有成熟治理的情况下快速推进 Agent 部署**——与 [[NIST AI Agent 安全标准|NIST 标准制定]]和 [[Gartner AI Agent 预测|Gartner 治理失败警告]]的背景高度一致。

74% 的企业预计两年内至少中等程度使用 Agentic AI——这与 [[Gartner AI Agent 预测|Gartner 的 40% 企业应用预测]]形成互补验证。

这个数字与 [[Fortune 500 企业 AI Agent 部署|Microsoft Cyber Pulse 报告]]的"80%+ Fortune 500 部署活跃 AI Agent"形成对比——差异在于定义和企业规模。

## 关键洞察

Deloitte 的数据是理解 [[AI Agent 市场规模|Agent 市场]]最重要的"清醒剂"之一。当 [[Gartner AI Agent 预测|Gartner 预测]] 2026 年底 40% 企业应用将集成 AI Agent 时，Deloitte 的 11% 生产部署率意味着需要在一年内实现约 **4 倍增长**——这不是不可能，但需要安全标准落地（[[NIST AI Agent 安全标准|NIST]]）、治理框架成熟（[[OpenClaw Foundation 治理|基金会化]]）、以及 [[OpenClaw 投资风险因素|安全事故]]频率下降。Karpathy 的判断可能更接近现实：Agent 的成熟需要的是一个"十年"，而非一年。
