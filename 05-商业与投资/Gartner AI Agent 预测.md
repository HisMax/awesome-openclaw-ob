# Gartner AI Agent 预测


![[assets/gartner-predictions.jpg]]

## 一句话总结

Gartner 预测到 2026 年底 40% 的企业应用将集成 AI Agent（2025 年不到 5%，增长 8 倍），到 2035 年 Agentic AI 可能驱动约 30% 企业应用软件收入，超过 $4500 亿。

## 核心数据

| 预测 | 数据 |
|------|------|
| 2025 年企业应用集成 AI Agent | **不到 5%** |
| **2026 年底预测** | **40%**（增长 8 倍） |
| 2027 年 Agent 项目失败率 | **40%** |
| 2035 年 Agentic AI 驱动的收入 | **$4500 亿+** |
| 2035 年企业软件收入占比 | 约 **30%** |
| 当前已部署 AI Agent 的组织 | **17%**（2026 年更新数据） |
| 预计两年内部署 AI Agent | **60%+** |
| 2028 年品牌使用 Agentic AI | **60%**（一对一互动） |
| 2029 年 IT 基础设施 Agent 部署 | **70%**（vs 2025 年 <5%） |
| 2026 全球 AI 支出增长 | **47%** |

## 分析

Gartner 的预测展现了 AI Agent 市场的两极：极度乐观的增长预期与极高的失败率并存。

Gartner 的 40% 预测建立在[[Agent 范式转变]]已发生的前提上——从"工具辅助"到"自主执行"的质变是企业大规模采用的先决条件。从 5% 到 40% 的 8 倍增长需要在一年内完成——对照 [[Deloitte 企业 AI Agent 采用报告|Deloitte 2026 报告的 23% 中等采用率]]，这意味着大量试点项目需要在 2026 年内转为生产部署。驱动力可能来自 [[Claude Code 营收分析|Claude Code 的企业渗透]]（50%+ 企业订阅）、[[Fortune 500 企业 AI Agent 数据|Fortune 500 的 80%+ 采用率]]，以及 [[AI Agent 市场规模|市场本身 40-50% 的 CAGR]]。

但 40% 的 Agent 项目失败率预警同样值得重视。Gartner 2026 年 5 月进一步警告：**对 AI Agent 施加统一治理将导致企业 Agent 项目失败**——Agent 需要差异化治理策略。失败的原因与 [[OpenClaw 投资风险因素|OpenClaw 暴露的安全问题]]高度相关：安全治理不足（[[Fortune 500 企业 AI Agent 数据|仅 47% 实施安全控制]]）、提示注入无法根本解决、以及 Agent 自主行为的不可预测性。

**2026 年新增关键预测**：
- Agentic AI 已进入 Gartner Hype Cycle 的**期望膨胀期顶峰**
- 到 2029 年，70% 的企业将部署 Agentic AI 来运维 IT 基础设施
- 2026 年全球 AI 支出预计增长 47%

$4500 亿的 2035 年预测将 Agentic AI 定位为企业软件的核心驱动力——从"辅助工具"变成"收入引擎"。

## 关键洞察

Gartner 的两个数字放在一起看最有意义：40% 的采用率 + 40% 的失败率 = **巨大的洗牌空间**。对[[商业化路径|投资者]]而言，这意味着 Agent 赛道的赢家不是"谁先部署"，而是"谁部署后能持续运行而不出事"。[[NIST AI Agent 安全标准|NIST 标准制定]]、[[OpenClaw Foundation 治理|开源治理]]、以及 Snyk 等[[OpenClaw 生态系统商业机会|安全工具]]提供商将成为这个转型过程的关键基础设施。$4500 亿的 2035 远景是真实的市场机会，但通往那里的道路将充满 [[Deloitte 企业 AI Agent 采用报告|试点失败]]和[[OpenClaw 投资风险因素|安全事故]]。
