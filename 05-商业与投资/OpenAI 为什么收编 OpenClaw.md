# OpenAI 为什么收编 OpenClaw

Sam Altman 在 2026 年 2 月 15 日公开表态（详见 Twitter 公开信）：
> "The future is going to be extremely multi-agent and it's important to us to support open source as part of that."

## 四大战略动机

1. **用户入口**：[[OpenClaw 是什么|OpenClaw]] 是目前最成功的"AI Agent -> 普通用户"桥梁，实现了编程民主化的愿景
2. **模型消耗**：237K stars 的项目意味着巨大的 API 调用量。但收编的核心是人——[[Peter Steinberger]] 既有将开源项目商业化的经验（PSPDFKit 退出价约 1 亿欧元），又掌握了 Agent 生态的关键技术理解，这才是 Altman 真正想要的。
3. **生态卡位**：5,705 个技能的生态系统 = 开发者粘性（见 [[Skills 市场]]），且与 MCP 协议生态形成互补
4. **竞争防御**：如果被 Google/Meta 获取，将是一个有力的分发渠道

## 隐含估值信号

- PSPDFKit（Steinberger 之前的公司）退出价约 1 亿欧元（2023）
- Steinberger 在 OpenAI 的薪酬"六位数以上"
- Claude Code 年化营收 $25 亿+（对比参考），Anthropic 整体 ARR 达 $470 亿（见 [[Anthropic Series H 融资]]）
- [[Cognition Series D 融资|Cognition（Devin）]]以 $260 亿估值证明 AI 编码 Agent 的独立商业价值
- Microsoft Build 2026 基于 OpenClaw 推出企业 Agent Scout——OpenClaw 的生态价值获得大厂级验证
- OpenClaw 的生态系统已经有公司盈利（见 [[OpenClaw Wrapper 创业潮]]），采用 MIT License 最大化了生态扩张
