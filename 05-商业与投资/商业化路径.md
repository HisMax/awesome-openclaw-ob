# 商业化路径

## 一句话理解

OpenClaw 本身不赚钱（MIT 开源、零收入），但它像一块磁铁，吸引了一整个商业生态在它周围生长——托管服务、安全产品、LLM 路由、教育内容。就像 Linux 本身免费，但围绕它的 Red Hat 价值数十亿。这正是 [[开源AI运动]] 的典型路径——但商业化和开源精神的张力始终是 OpenClaw 最大的治理挑战：一旦商业利益主导，社区信任就会流失；而纯粹开源又无法支撑长期发展。

## OpenClaw 自身的商业模式：没有

| 收入来源 | 状态 |
|----------|------|
| 软件售卖 | 免费 |
| 订阅费 | 无 |
| 企业版 | 无 |
| 广告 | 无 |
| 基金会赞助 | OpenAI + 其他 |

这是一个**纯公共品**——创始人 [[Peter Steinberger]] 说："I don't do this for the money." 他最终加入了 OpenAI，项目转交独立基金会。

## 用户实际成本

虽然软件免费，但运行一个 Agent 需要基础设施。最大的成本项是 LLM API 调用费：

| 使用强度 | 月费估算 |
|----------|----------|
| 轻度（本地 + Haiku） | $1-5 |
| 中度（VPS + Sonnet） | $20-50 |
| 重度（VPS + Opus） | $100-200 |
| 极端（多 Agent + 重度 API） | $400-700+ |

详细成本拆解见 [[API 定价与成本分析]]。

## 围绕 OpenClaw 的五大商业方向

### 1. 托管/部署服务

把"技术活"变成"一键服务"，面向不会运维的用户：

| 公司 | 模式 |
|------|------|
| SetupClaw | 白手套部署服务，面向 4-50+ 人团队 |
| DigitalOcean | 1-Click Deploy 镜像 |
| Cloudflare | moltworker（Workers 上运行） |
| Fly.io | 容器化部署 |

### 2. 安全服务

安全问题催生了安全产品需求：

- **Snyk**：Skill Security Scanner + Agent Trust Hub
- **Daytona**：安全沙箱运行环境

### 3. LLM 基础设施

[[OpenRouter 融资与商业|OpenRouter]] 是典型代表——所有推理支出收取 5% 佣金，估值约 $5 亿，接入 400+ 模型。

### 4. Wrapper 创业潮

TrustMRR 追踪 117 家基于 OpenClaw 的创业公司：

- **SimpleClaw**：上线 2 周达 $30K MRR
- **OpenClawd**：类比 "Vercel 之于 Next.js"
- **ClawBox**：SaaS 订阅制托管

> 但壁垒极低——多家 wrapper 公司已在 TrustMRR 挂牌出售。一旦官方推出云版本，90% 的 wrapper 可能失去生存空间。

### 5. 内容/教育

| 创作者 | 成果 |
|--------|------|
| Nat Eliason | $4K/周收入的创业教程 |
| Ken Yeung | ClawBeat.co 新闻聚合站 |
| freeCodeCamp | 完整入门教程 |

### 大厂争夺

Sam Altman 公开表态："The future is going to be extremely multi-agent." OpenAI 收编了 Peter Steinberger，Mark Zuckerberg 也亲自联系过他。Anthropic 从 [[Anthropic Series G 融资|Series G（$3,800 亿估值）]]到 [[Anthropic Series H 融资|Series H（$9,650 亿估值）]]三个月翻 2.5 倍，ARR 达 $470 亿。Microsoft 在 Build 2026 直接基于 OpenClaw 构建企业 Agent"Scout"。[[Cognition Series D 融资|Cognition（Devin）]]以 $260 亿估值成为 AI 编码 Agent 赛道新星。这说明大厂都把 AI Agent 视为核心战略赛道。

详细分析见 [[OpenAI 为什么收编 OpenClaw]]。

## 与 Anthropic 的微妙关系

OpenClaw 最初名为 **Clawdbot**（Claude 的双关语），Anthropic 发出商标通知迫使改名。创始人最终加入了 OpenAI 而非 Anthropic。这是一个值得注意的竞争动态——Anthropic 的商标行动可能间接将一个强大的生态推向了 OpenAI 阵营。

详见 [[OpenClaw 与 Anthropic 关系]]。

## 核心洞察

OpenClaw 的商业化路径证明了一个规律：**真正有价值的开源项目不需要自己赚钱——它创造的生态会替它变现**。问题在于这个生态是否可持续，还是只是一波短暂的"淘金热"。[[OpenClaw Wrapper 创业潮]] 的高流转率暗示后者的可能性不容忽视。
