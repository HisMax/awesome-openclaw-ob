# OpenClaw 商业模式


![[assets/business-model.jpg]]

OpenClaw 采用**完全免费开源**策略，MIT 许可证，项目本身不产生收入。这是由 [[Peter Steinberger]] 创立时确定的核心策略。

## 收入来源分析

| 收入来源 | 状态 |
|----------|------|
| 软件售卖 | 无 |
| 订阅费 | 无 |
| 企业版 | 无 |
| 广告 | 无 |
| Bug 赏金 | 无 |
| 基金会赞助 | OpenAI + 其他 |

唯一的资金来源是 [[OpenClaw Foundation 治理]] 接受的赞助。

## 用户成本结构

| 组件 | 费用 |
|------|------|
| OpenClaw 软件 | 免费 |
| VPS 托管（推荐） | 5-28 欧元/月 |
| LLM API 调用 | $5-200+/月 |
| 本地模型（Ollama） | $0（需硬件） |

## 典型用户月费

- **轻度使用**：$1-5/月（本地 + Haiku）
- **中度使用**：$20-50/月（VPS + Sonnet）
- **重度使用**：$100-200/月（VPS + Opus）
- **极端案例**：$400-700+/月（多 Agent + 重度 API）

项目的商业模式缺失是 [[OpenClaw 投资风险因素]] 之一，但其免费开源特性也是 [[OpenClaw 投资机会因素]] 的核心优势，最大化了用户基础。

围绕 OpenClaw 形成了丰富的 [[OpenClaw 生态系统商业机会]]，包括托管服务、安全服务和创业公司。

## 2026 Q2 商业化新动态

OpenClaw 的商业化路径在 2026 年 Q2 出现了重大变化：

1. **Microsoft Scout**：微软在 Build 2026 上发布基于 OpenClaw 的企业 Agent"Scout"，并承诺将安全防护代码回馈上游。这是 OpenClaw 从"个人工具"到"企业基础设施"的标志性事件
2. **NVIDIA NemoClaw**：NVIDIA 宣布 NemoClaw 方案，将 OpenClaw 接入 NVIDIA AI 计算栈
3. **GitHub Stars 373K+**：社区持续高速增长，贡献者超 1,200 人
4. **[[Anthropic Series H 融资]]**：Anthropic $470 亿 ARR 的成功，反证了 AI Agent 生态的商业价值
5. **[[Cognition Series D 融资]]**：Devin $4.92 亿 ARR 展示了 AI 编码 Agent 的直接变现能力
