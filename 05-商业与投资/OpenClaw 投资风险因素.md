# OpenClaw 投资风险因素


![[assets/investment-risk.jpg]]

## 主要风险

| 风险 | 说明 |
|------|------|
| **安全债务** | 无安全团队、无 Bug 赏金、多个严重 CVE（详见 [[OpenClaw 安全风险]]） |
| **创始人风险** | [[Peter Steinberger]] 已加入 OpenAI，社区治理能力存疑 |
| **Stars 争议** | 增长数据可能被夸大（详见 [[OpenClaw 社区热度总览]]） |
| **商业模式缺失** | 项目本身不产生收入（详见 [[OpenClaw 商业模式]]） |
| **监管风险** | 荷兰 AP 已警告 GDPR 合规问题 |
| **竞争加剧** | Microsoft Scout（基于 OpenClaw）、Claude Cowork、[[Cognition Series D 融资\|Devin $260 亿估值]]等大厂/独角兽全面入场 |
| **基金会治理空白** | 详见 [[OpenClaw Foundation 治理]] |
| **大厂"拥抱-吞噬"风险** | Microsoft Scout 直接基于 OpenClaw 构建，可能稀释社区独立性 |

## Wrapper 创业可持续性警告

多家 [[OpenClaw Wrapper 创业潮|wrapper 创业公司]] 已同时在 TrustMRR 挂牌出售，创始人套现离场。壁垒极低，一旦 OpenClaw 官方推出云版本，90% 的 wrapper 可能失去生存空间。[[Skills 商业化与定价]] 的不确定性也增加了生态商业化的风险。

## 2026 Q2 新增风险变量

- **Anthropic IPO 冲击**：[[Anthropic Series H 融资|Anthropic 估值 $9,650 亿]]并秘密提交 S-1，IPO 成功将进一步巩固 Claude Code 的企业护城河，压缩 OpenClaw 的商业空间
- **Gartner 治理警告**：Gartner 2026 年 5 月预测对 AI Agent 施加统一治理将导致项目失败——OpenClaw 的零治理策略既是风险也是灵活性
- **Devin 的 89% 自编码**：[[Cognition Series D 融资|Cognition 展示]] AI Agent 能写自己 89% 的代码，暗示 OpenClaw 的开发者生态优势可能被 AI 自动化削弱

## 对冲视角

风险需与 [[OpenClaw 投资机会因素]] 综合评估。Gartner 数据显示目前 17% 组织已部署 AI Agent（较此前 8% 有明显提升），市场正从早期阶段加速进入增长期。安全问题若得到系统性解决，将显著降低风险。
